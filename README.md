# Python Calculator 🧮

This is a simple calculator built with Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous calculations
- Error handling

## How to Run
1. Open terminal
2. Run:
   python calculator.py

##Future Improvements

During development, I encountered challenges with input validation and program flow. Handling invalid user input (e.g., non-numeric values) initially caused runtime errors, which I resolved using try-except blocks.

I also found it challenging to implement continuous calculations without resetting the result after each operation. This required a better understanding of loops, variable scope, and control flow. Overcoming these issues helped strengthen my understanding of Python fundamentals.
